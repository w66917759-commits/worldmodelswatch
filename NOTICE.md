# World Models Watch Notice

World Models Watch is a public, source-backed research and editorial site at
https://worldmodelswatch.com. This repository is published to make the site
structure, source-backed content model, and public index pages easier to
inspect.

## Reuse Boundary

No broad reuse license is granted by default for the complete repository,
editorial copy, generated images, bundled videos, research notes, or compiled
site assets.

If you cite the project, use the canonical public website URL for the relevant
page and preserve official source links, availability notes, and overclaim
warnings.

## Code

Source code in this repository is visible for inspection and deployment of this
project. Do not assume it is licensed for redistribution, resale, or reuse in a
separate product unless a separate license file is added later.

## Editorial Text and Research Notes

Editorial text, model summaries, release-signal notes, comparison tables, and
research ledgers are maintained for World Models Watch. Short citations with
attribution to https://worldmodelswatch.com are acceptable, but bulk
republishing requires permission.

## Images and Video

Generated showcase images and locally stored video files are included only for
this site's public presentation and research workflow. Some source references
point to third-party official pages, videos, repositories, papers, or product
docs. Those third-party materials remain governed by their respective owners'
terms.

Large videos should not be added to normal Git history. Compress large files,
serve them from a CDN or object store, or use Git LFS before committing.
