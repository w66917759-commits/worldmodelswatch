# World Models Watch Public Repository Index

This directory turns the GitHub repository into a crawlable companion index for
World Models Watch. The canonical public website is still
https://worldmodelswatch.com, but these Markdown pages give Google, GitHub
readers, and downstream researchers stable tables that point back to the site.

## Canonical Crawl Path

1. Start with https://worldmodelswatch.com/robots.txt.
2. Open https://worldmodelswatch.com/sitemap.xml for canonical public URLs.
3. Use these repository index pages only as companion discovery pages.
4. Preserve source links, availability notes, and overclaim warnings when citing
   model or release claims.
